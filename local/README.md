# Local-only artifacts

Thư mục này dùng để giữ tài liệu học tập riêng tư hoặc tạm thời trên máy local.

Các loại nội dung nên đặt tại đây:

- Câu hỏi tự học.
- Câu trả lời nháp.
- Review câu trả lời.
- Ghi chú mentor/reviewer.
- Prompt thô.
- Scratch notes.
- Bản backup trước khi tái cấu trúc repo.

Theo chính sách `.gitignore`, mọi file thật trong `local/` đều bị ignore, ngoại trừ file README này.
